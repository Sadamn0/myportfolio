## Getting Started

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. Customize the content with your information

## Customization

1. Update your name in the hero section
2. Replace contact information (email, phone, location)
3. Add your actual projects and links
4. Update social media links
5. Modify skills and percentages
6. Add your profile photo (replace the placeholder)
